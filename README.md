# Bike Rental

A Frappe-based bike rental management system for multi-hub operators.

## Features

- Multi-hub fleet inventory management
- Model-based booking with real-time availability
- Overbooking prevention with row-level locking
- Ghost-booking auto-expiry
- Check-out/check-in with KM and battery tracking
- Dynamic pricing (weekend/peak season overrides)
- Customer KYC verification
- Staff dashboard and reporting
- Email and SMS notifications
- One-way rentals and inter-hub transfers
